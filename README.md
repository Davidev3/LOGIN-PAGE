A cinematic login experience — particles, blur, mountains.

✨ Features

🌌 Animated particle canvas — 100 floating particles rendered with the Canvas API
🪟 Glassmorphism card — backdrop-filter: blur with frosted glass aesthetics
🎞️ Fade-in animation — smooth page entrance via CSS keyframes
🔔 Custom popup feedback — success/error messages with auto-dismiss
📐 Responsive layout — centered with flexbox, works on any screen

🎨 Design Decisions
The background is a purple mountain twilight scene — chosen to contrast beautifully with the white frosted glass card. The particles add depth without cluttering the UI. The color palette leans into purple/blue tones for a dark, immersive atmosphere.
