## ✨ Features

- 🌌 **Particle Canvas** — 100 animated particles via Canvas API + `requestAnimationFrame` loop
- 🪟 **Glassmorphism Card** — `backdrop-filter: blur` with frosted glass aesthetics and rgba borders
- 🎞️ **Fade-in Animation** — smooth page entrance via CSS `@keyframes` on load
- 🔔 **Custom Popup** — success/error feedback with auto-dismiss after 2.5s
- 📐 **Responsive Layout** — centered with flexbox, works on any screen size

## 🚀 Quick Start

## 🧠 Concepts Used

| Concept | Where | Type |
|---|---|---|
| Canvas API + `requestAnimationFrame` | Particle animation loop | JS API |
| `backdrop-filter: blur()` | Glassmorphism card | CSS |
| CSS `@keyframes` | Page fade-in on load | CSS |
| DOM manipulation | Popup show/hide logic | JS |
| `e.preventDefault()` | Form submission control | JS |
| Flexbox | Centering & responsive layout | CSS |

## 🎨 Design

The background is a **purple mountain twilight scene**, chosen to contrast with the white frosted glass card. Floating particles add depth without cluttering the UI. The palette leans into purple/blue tones for a dark, immersive atmosphere.
